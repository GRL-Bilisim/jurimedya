Media Object

Abstract object styles for building various types of components that feature a left or right aligned image alongside textual content.

Left Aligned Media Objects

Wrap your <img> tag inside of .media class to align your image to left. You don't have to add any extra classes to align your text to left.

Media Alignment

The images or other media can be aligned top, middle, or bottom. The default is top aligned. Now, for vertical align, Top alignment use .align-self-start class. For Middle alignment, use .align-self-center and for Bottom alignment, use .align-self-end with left or right aligned class.
